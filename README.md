# Master-in-Autonomous-Vehicle-Systems
Roadmap for mastering autonomous vehicle systems

## Core Foundations
### Topics to cover

1. **Robotics Mathematics**: Linear algebra, calculus, transformations, geometry
2. **Probability & Statistics**: Uncertainty modeling, distributions, Bayesian thinking
3. **Control Theory**: PID control, state-space models, model predictive control (MPC)
4. **Vehicle Motion Principles**: Dynamics, kinematics, traction, steering models

## Perception
**Computer Vision & Machine Learning**

1. Image processing with OpenCV
2. Deep learning for detection, tracking, and segmentation
3. CNNs, transformers, 3D vision basics

**Sensors**
1. LiDAR
2. Radar
3. Stereo/mono cameras
4. Depth cameras
5. IMU, GNSS

**Sensor Fusion**
1. Kalman Filters (KF)
2. Extended KF (EKF)
3. Unscented KF (UKF)
4. Multi-sensor fusion architectures and pipelines


## Localization & Mapping
**Key Modules**
1. SLAM: LiDAR SLAM, Visual SLAM, RGB-D SLAM
2. Odometry: Wheel, visual-inertial odometry
3. Mapping: HD map creation, map interpretation, semantic maps
   
## Planning & Decision Making
**Route and Behavior Planning**
1. Motion planning algorithms: A*, D*, RRT, lattice planners
2. Behavior planning: finite state machines, behavior trees
3. Trajectory generation, smoothing, and optimization

## Systems & Software/Middleware
1. ROS / ROS2: nodes, topics, actions, services, launch files
2. Simulation: Omniverse, Gazebo
3. Visualiztion tools : RViz / RViz2
